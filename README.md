# Personal Website

Lightweight, static personal homepage (plain HTML/CSS/JS).

## Local preview

From this folder, run:

```sh
python3 -m http.server 8000
```

Then open `http://localhost:8000/` in your browser.

## Customize

- Edit `index.html` (your content)
- Replace `images/logo-bright.png` and `images/logo-dark.png` (your logo/photo)
